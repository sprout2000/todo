## 使い方

```sh
# このレポジトリをクローン
git clone https://github.com/sprout2000/todo.git

# 依存パッケージをインストール
cd todo
npm install

# http://localhost:5173 で開発
npm run dev
```

## ユニットテスト

```sh
npm test
```
